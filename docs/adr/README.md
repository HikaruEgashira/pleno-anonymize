## ADRフォーマット

各ADRは以下の構造に従います:

- **Status**: Proposed / Accepted / Deprecated / Superseded
- **Context**: 決定が必要な背景
- **Decision**: 何を決定したか
- **Consequences**: 決定による影響（Positive/Negative）
